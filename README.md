# Judith Navó — Portfolio 3D

Portfolio interactivo en 3D hecho con Three.js. Una habitación con monitores y objetos clicables.

---

## Cómo abrirlo

No abrir directamente el archivo — usar un servidor local:

- **VS Code**: instala la extensión **Live Server**, clic derecho en `index.html` → *Open with Live Server*

---

## Qué puedes hacer

- **Ratón** → rotar la cámara arrastrando
- **Scroll** → zoom
- **Hover** sobre objetos → aparece tooltip
- **Click en Pantalla01** → abre el portfolio web
- **Click en Pantalla02** → abre el sistema de archivos (CV, Skills, Sobre mí)
- **Click en artstation / linkedin / gmail** → abre el link
- **Botón LIGHTS** → apaga y enciende las luces
- **ESC o ✕** → cierra la pantalla abierta
